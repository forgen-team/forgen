## TL;DR (셀링 한 줄 — 학술 증명 수준)

> **forgen 단독 사용은 Claude/Codex 응답 품질을 통계적으로 유의하게 향상** —
> 양 driver retry+sequential N=33 합산 (pooled N=66) 기준 **mean δ = +0.151 W,
> 95% CI [+0.118, +0.184] (0 을 완전 제외), 62/66 (93.9%) 케이스 양수, sign
> test p = 1.04×10⁻¹⁴**. Codex N=33 단독 +0.176 (32/33 = 97% 양수, p =
> 4×10⁻⁹). Claude N=33 단독 +0.125 (30/33 = 91% 양수, p = 7×10⁻⁷). **세 독립
> 측정 모두 δ > 0 학술 수준 증명**. ψ master gate (forgen+mem coexistence) 는
> 양 driver 모두 mean ≈ 0, FAIL — **forgen+mem 결합 추가 효과는 noise 영역**
> 으로 확정, **forgen 단독 사용 권장**.

**ψ master gate 결론 (forgen+mem coexistence)**: 양 driver 모두 mean ≈ 0 (claude
−0.005, codex −0.021), CI 양쪽 모두 0 가로지름 → **결합 추가 효과는 통계적으로

W 는 0–1 범위의 효과 크기 (γ·β·δ·ε·ζ 합산, 자세한 정의 ADR-006). +0.046 ~
+0.156 은 응답 품질 5–16% 포인트 개선에 해당. **세 측정 모두 N 다르고 fix 다른
시점인데 부호와 다수 케이스 양수 일관**.

### 무엇을 셀링하지 않는가 (정직한 한계)

- **ψ (forgen+mem coexistence) 는 noise 영역**: claude driver +0.020 / codex
  driver −0.051, 부호 갈리고 CI 가 0 가로지름. **forgen+mem 동시 사용 의
  추가 효과는 측정 불가능 한 작은 영역.** 권장: forgen 단독 사용.
- **κ β agreement 약함**: judge 간 persona-specific axis 합의도 0.18–0.20
  (gamma 는 0.58–0.69 양호). β 채점은 더 많은 anchor / rubric 정제 필요.
- **N=10 small sample**: 향후 N=30 confirm 측정으로 CI 좁히기 + outlier 영향
  완화 권고.
