   - (a) content-hash가 manifest와 일치 → **무프롬프트 삭제** (forgen이 쓴 그대로)
   - (b) provenance 마커만 매치 (`<!-- forge-tuned -->`, `/^# Tenetx/`, `forge-generated v\d/`) + hash 불일치 → **프롬프트** (사용자 편집 가능성)

4. **백업**: 모든 삭제/편집 전 `~/.forgen/backups/reclaim-<ts>/` 이동. `--dry-run` 지원.
5. **Tests** (vitest, tmp-HOME fixture): hash-match 삭제 / marker-only 프롬프트 / 사용자 파일 불간섭 / settings 탐지 / idempotent 재실행 / dry-run 무변경. 2026-07-16 수동 청소에서 확보한 실제 tenetx 파일들을 fixture로 사용 (백업 디렉토리에 실물 있음).
